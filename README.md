# Projet : AUTHENTIFICATION

## Description du projet

### ETAPE 1: Simple authentification
Il s'agit de mettre en place le backend d'une authentification simple avec un email et un mot de passe.

#### Fonctionnalités abordées
  - Login
  - Logout
  - RefreshToken
  - Middleware authenticate

### ETAPE 2: Authentification à 2 facteurs
Dans ce cas, on teste une authentification à double facteur (le cas d'un TOTP).

#### Focntionalités abordées
  - Generation de secret
  - Generation de QRCODE
  - Activation de la 2FA
